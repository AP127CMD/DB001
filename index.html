Apps Script Relay — Live Data Connection

Google Sheets CSV cannot be fetched directly from a local HTML file (browser CORS policy). A Google Apps Script acts as a relay — it runs on Google's servers, fetches the sheet data, and returns it to this page with the correct headers. It's free and takes about 2 minutes to set up.

Step 2 — Go to script.google.com → New Project → paste the code → Save
